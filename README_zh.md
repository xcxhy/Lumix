# Lumix

这是一个准备大语言模型数据的开源项目，由于大家都在卷模型的预训练与指令微调，大部分公开的项目也很少提及处理清洗数据的细节。

我希望这个项目能够帮助到大家，尽可能完成数据的清洗工作，让大家能够更加专注于模型的训练与微调！
